# 🔐 API de Autenticação Segura com JWT

Backend robusto para gestão de usuários, utilizando as melhores práticas de segurança do mercado.

## 🛡️ Segurança Implementada
- **Criptografia (Bcrypt)**: As senhas nunca são salvas como texto simples no banco.
- **JWT (JSON Web Tokens)**: Geração de tokens para autenticação de rotas protegidas.
- **SQLite**: Banco de dados leve para persistência de credenciais.

## 🚀 Tecnologias
- **Node.js** & **Express**
- **Bcrypt**: Para hashing de senhas.
- **Jsonwebtoken**: Para gestão de sessões.

## ⚙️ Como rodar
1. `npm install`
2. `node server.js`
3. Use o `index.html` para testar o fluxo de cadastro e login.
